## 🧠 Overview

The **AI-Powered SAP Business Process Simulator** is a modern, interactive web application that simulates core enterprise workflows across SAP modules.

It demonstrates how different business processes — procurement, production, sales, and finance — are integrated and how data flows in real time to support decision-making.

Designed with a SaaS-style UI, the project combines **business logic + data visualization + modern frontend engineering**.

---

## 🎯 Key Modules Simulated

### 🧾 Procure-to-Pay (MM)
- Create purchase orders
- Increase raw material inventory
- Track procurement expenses

### 🏭 Plan-to-Produce (PP)
- Convert raw materials into finished goods
- Simulate production workflows
- Update inventory dynamically

### 💰 Order-to-Cash (SD)
- Create sales orders
- Generate revenue
- Reduce finished goods stock

### 📊 Record-to-Report (FI)
- Track expenses and revenue
- Calculate profit
- Visualize financial performance

## ⚙️ How It Works

The application simulates a complete business lifecycle:

1. **Procurement**
   - Purchase raw materials → increases inventory → increases expenses  

2. **Production**
   - Raw materials are consumed → finished goods are created  

3. **Sales**
   - Finished goods are sold → revenue is generated  

4. **Finance**
   - System tracks revenue vs expenses → calculates profit  

👉 All modules are interconnected, reflecting real-world SAP integration.
